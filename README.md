This repository contains the R Markdown and datafiles used for producing the figures in the manuscript 'Connectivity of terrestrial deep biosphere microbial biomes with surface communities', written by:
George Westmeijer, Maliheh Mehrshad, Stephanie Turner, Linda Alakangas, Varvara Sachpazidou, Carina Bunse, Jarone Pinhassi, Marcelo Ketzer, Mats Åström, Stefan Bertilsson, & Mark Dopson. The R script relies on the ASV-table ('seqtab.tsv'), the taxonomy table ('taxa.tsv') and the sample metadata ('smd.tsv'). Additional data files are the
number of reads per sample throughout the bioinformatic pipeline ('data_file_s1.tsv'), the composition of the core community as defined in the manuscript ('data_file_s2.tsv') and finally the compositon of the accessory community ('data_file_s3.tsv').
